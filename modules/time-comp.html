    • Which Algorithm is faster ?
  • Time taken for a function to complete its task depends on various factors
  • How to measure the Speed of a Function ?
    • Time Taken (in sec.)
      • But that would change depending on the input data set size.
      • Also on the machine which it's being executed.
    • Number of statements executed.

Why we need Time Complexity ?
  • To make it Independent of the mentioned dependencies.
  • To understand the Time taken only based on Input Size
Big O Noation
  • This is a way to measure how well our Algorithm scales as the amount of data increases
  • Ex 1.: Input set of 10 elements compared to input set of million elements
  • Ex 2.: Which element has the biggest effect on the answer: Q : 2*n^3 + 5*n^2 + 19
    • when n = 2, the answer is 2*8 + 5*4 + 19 = 55
    • when n = 3, the answer is 2*27 + 5*9 + 19 = 118
    • when n = 10, the answer is 2*1000 + 5*100 + 19 = 2519
      • As we can see, element - 19 is insignificant as n increases
      • In fact, n^2 is alse dwarfed by n^3 as n increases
      • n^3 is the main contributor (even the constant, 2, doesn't contribute much)
      • Hence, the Complexity of this question is O(n^3)
